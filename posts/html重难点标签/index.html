Html重难点标签

January 1, 2021

html的重难点标有:a标签，table标签，img标签，form标签，input标签

a标签

a标签的属性有:

  • href
  • target
  • download
  • rel=noopener

a标签的作用: 1、跳转到外部页面 ，2、跳转到内部锚点，3、跳到邮箱或者电话

a的href取值:
网址:

https://google.com

http://google.com

//google.com

路径

/a/b/c(绝对路径)以及a/b/c(相对路径)，index.html及./index.html

伪协议

javascript:代码;

maito:邮箱

tel:手机号

id

href=#xxx

a标签的target取值:

_blank:在新窗口打开

_top:在顶部，最外面的窗口打开

_parent:在父级窗口打开

_self:基于当前窗口打开

table标签

img标签

作用:发出get请求，展示一张图片。

属性有:alt/height/width/src,其中alt表示:图片加载失败时提示文字

相关的事件:

onload:图片加载成功时

onerror：图片加载失败时会触发

我们来看一段代码:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
     <img src="dog.png" alt="我是二哈" id="xxx" height="400px">
     <script>
         xxx.onload=function(){
             console.log('图片加载成功')
         }
         xxx.onerror=function(){
             console.log('图片加载失败')
             xxx.src="404.png"
         }
     </script>
</body>
</html>

最后图片加载成功，控制台打印出"图片加载成功" 响应式

代码如下:max-width:100%

我们来看一段将图片设置成响应式的代码:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        *{
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }
       img{ width: 100%; max-width: 100%; }
        
    </style>
</head>
<body>
    <img src="./404.png" alt="我是二哈">
</body>
</html>

form 标签

作用:发出get或post请求，然后刷新页面

属性 :action/aotocomplete/method/target

action:请求的页面地址

autocomplate:自动填充name(要设置为on,并且有name=“username"才生效)

事件 ：onsubmit

input的submitbutton的submit有何区别?

input只能有纯文本，不能有其他标签，button可以有其他标签

来看一段代码:

<body>
    <form action="/yyy" method="POST" autocomplete="on" target="a">
      <input type="text" name="username">
      <input type="submit" value="<strong>搞起</strong>">
      <button type="submit"><strong>提交</strong> <img src="dog.png" width="100px" alt=""></button>
    </form>
</body>

可以看出input里的strong标签并没有生效

input标签

作用 :让用户输入内容

属性type：button/checkbox/email/file/hidden/number/password/radio/search/submit/tel/text

其他name :autofocus/checked/disabled/maxlength/pattern/value/placeholder

事件:onchange/onfocue/onblur

其他输入标签:select+option/textarea/label

我们通过一个小练习来学习它的用法吧:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <form action="/yyy" method="POST" autocomplete="off" target="a">
       <input type="text" required/>
       <hr/>
       <input type="color"/>
       <hr/>
       <input type="password"/>
       <hr/>
       <input type="radio" name="gender">男
       <input type="radio" name="gender">女
       <hr/>
       <input type="checkbox" name="hobby">唱
       <input type="checkbox" name="hobby">rap
       <input type="checkbox" name="hobby">跳
       <input type="checkbox" name="hobby">篮球
       <hr/>
       <input type="file"/>
       <input type="file" multiple/>
       <hr/>
       看不见我吧: <input type="hidden" value="hidden">
       <hr/>
       <textarea style="resize: none;width: 50%;height: 300px;"></textarea>
       <hr/>
       <select>
           <option value="">-请选择-</option>
           <option value="1">星期一</option>
           <option value="2">星期二</option>
       </select>
    </form>
</body>
</html>
